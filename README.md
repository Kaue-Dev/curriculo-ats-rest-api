# Currículo ATS API

Projeto base em **Node.js** com **Fastify**.

## Scripts

- `npm install` – instala as dependências.
- `npm run dev` – sobe o servidor em modo desenvolvimento com `node --watch`.
- `npm start` – sobe o servidor em modo produção.

## Endpoints iniciais

- `GET /health` – retorna `{ "status": "ok" }`.
